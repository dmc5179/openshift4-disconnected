# Ansible role 'mirror_rhcos'

Ansible role for mirroring the Red Hat CoreOS install media.
Role includes variables for mirroring RHCOS as:
- VMDK
- ISO
- Azure VD
- GPE

## Requirements

- No Requirements

## Dependencies

- No Dependencies
